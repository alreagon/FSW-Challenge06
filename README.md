# FSW-Challenge06

## Getting Started

1. Clone the repo

   ```sh
   git clone https://github.com/alreagon/FSW-Challenge06
   ```

2. Install packages
   ```sh
   yarn install
   ```
3. Next Step

   ```sh
   setting config/config.js with your database, in my case i use xampp/mysql
   ```

4. Make Database

   ```sh
   yarn run create
   ```

5. Migrate Table Database

   ```sh
   yarn run migrate
   ```

6. Seeders

   ```sh
   yarn run seed
   ```

7. Start The Project

   ```sh
   yarn start
   ```
